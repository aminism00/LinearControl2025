# پروژه کنترل خطی – سیستم تانک آب

## معرفی
در این پروژه، مدل‌سازی و طراحی کنترل‌کننده خطی برای سیستم تانک آب انجام شده است. هدف، کنترل سطح آب و تنظیم آن روی مقدار مرجع با عملکرد پایدار و خطای کم است. مدل‌سازی، طراحی کنترل و شبیه‌سازی در محیط MATLAB/Simulink انجام شده است.

## فایل‌های پروژه

### فایل‌های شبیه‌سازی
- MyModel.slx → مدل اصلی سیمولینک  

### کدهای MATLAB
- State_Space.m → پیاده‌سازی مدل فضای حالت  
- myInput.m → تعریف ورودی سیستم  
- mydata.mat → داده‌های مورد استفاده در شبیه‌سازی  

### گزارش و مستندات
- main.pdf → گزارش اصلی پروژه  
- model.pdf → توضیحات مدل‌سازی  
- controllerModel.pdf → طراحی کنترل‌کننده  
- noise-1.pdf / noise-2.pdf → تحلیل اثر نویز  

### سایر فایل‌ها
- PowerPoint.zip → فایل ارائه پروژه  
- pr.zip → نسخه فشرده پروژه  


## نتیجه
کنترل‌کننده طراحی‌شده توانسته است سیستم را پایدار کرده و سطح آب را با خطای کم به مقدار مرجع برساند.
